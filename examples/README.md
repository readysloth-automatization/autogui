# Как конфигурировать

Конфиге GUI Automation содержит:
 - Общие параметры:
  ```
    "execute" : "freedoom1",    -- имя исполняемого файла
    "resolution" : "1920x1080", -- разрешение экрана для xvfb
    "initial timeout" : 1,      -- задержка в секундах перед
                                   стартом всех тестов (может быть дробной)
    "timeout between tests" : 0,-- задержка между выполнением каждого теста
    "confidence" : 0.9,         -- точность определения изображения
  ```
 - Очередность выполнения тестов задается
 следующим списком:
  ```
    "perform" : [
        "get to menu", -- название теста
        "find enemy"
    ],
  ```
 - Список тестов:
  ```
    "tests" : [
  ```
   - В свою очередь список тестов содержит:
      ```
        {
            "do": "get to menu", -- название теста
            "timeout" : 2,       -- задержка перед исполнением
                                    конкретного теста.
                                    Суммируется с "timeout between tests".
                                    
            "steps": [           -- шаги для выполнения теста.
                "press esc",        Шаг может указывать на другие
                "press enter",       + тесты,
                {                    + вызывать методы pyautogui
                "method" : "press",
                "argument": "'enter'",
                "timeout" : 2
                },
                "xcalc/click_4.png", + кликать на элемент интерфейса
                {                    + или иметь в себе python код,
                                       для специфических целей.
                                       В таком случае имя метода должно
                                       быть HANDMADE

                "method" : "HANDMADE",
                "argument": "pyautogui.keyDown('w'); time.sleep(2); pyautogui.keyUp('w')",
                "timeout" : 0
                }

            ]
        },
      ```
